# Jarvis

Jarvis - my personal assistant. 

## Installation

```bash
pip install -e .
```

For development dependencies:

```bash
pip install -e ".[dev]"
```

## Usage

```bash
jarvis
```

## Development

Run tests:

```bash
pytest tests/
```
